# perl-Math-BivariateCDF

Perl wrapper for Bivariate CDF function.

Installation
============

To install this module, run the following commands:

        perl Makefile.PL
        make
        make test
        make install

Dependencies
============

* https://people.sc.fsu.edu/~jburkardt/c_src/toms462/toms462.html
